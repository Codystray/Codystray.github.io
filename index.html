GitHub ↗

A key-value storage engine written in C leveraging the new NVMe Key-Value Command Set. Focuses on low-latency I/O, efficient memory management, and a clean storage abstraction layer. Collaborative senior capstone project with a focus on performance and correctness.

C Systems Programming SPDK / KVSSD Storage Engine Capstone Project

Xv6 Kernel Extensions
GitHub ↗

A series of kernel-level labs extending Xv6, a Unix-like teaching OS running on RISC-V. Labs cover system calls, page tables, traps, copy-on-write fork, locks, file systems, and memory-mapped I/O. All kernel modifications written in C and tested in QEMU.

C Kernel Development RISC-V QEMU / GDB Operating Systems II
